# Tchat Client Standalone

## Technologies

### Tools

- Node.js
- Typescript
- TSD DefinitelyTyped

### Application

- Angular 2 (beta 3)
  - RxJS 5 (beta 0)
- Bootstrap 4 (alpha 2)

### Tests

- Jasmine
- Karma
- Wallaby.js

## Installation

>`npm install`

>`tsd install`

## Execution

### Start server

>`gulp serve`

Lunch browser `http:\\localhost:3000`

### Test

Lunch wallaby.js on your editor
